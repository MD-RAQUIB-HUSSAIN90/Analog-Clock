# 🕒 Analog Clock (Responsive Web App)


<h3>LIVE DEMO:- </h3>
## 📌 Description
This is a simple and responsive **Analog Clock** built using HTML, CSS, and JavaScript.  
It displays real-time hours, minutes, and seconds with smooth rotation of clock hands.

The design is fully responsive and works perfectly on mobile, tablet, and desktop devices.

---

## 🚀 Features
- Real-time analog clock
- Smooth rotating clock hands
- Fully responsive design
- Mobile, tablet, and desktop friendly
- Clean and minimal UI

---

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript

---

## 📱 Responsive Design
The clock automatically adjusts its size using modern CSS techniques like:
- `vw` units
- `min()` function
- `aspect-ratio`

So it looks perfect on all screen sizes.
